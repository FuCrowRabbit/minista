# minista-plugin-enhance

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-enhance
```

```js
// ./minista.config.js
import { pluginEnhance } from "minista-plugin-enhance"

export default {
  plugins: [pluginEnhance()],
}
```
