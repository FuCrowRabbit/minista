# minista-plugin-mdx

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-mdx
```

```js
// ./minista.config.js
import { pluginMdx } from "minista-plugin-mdx"

export default {
  plugins: [pluginMdx()],
}
```
