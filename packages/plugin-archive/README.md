# minista-plugin-archive

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-archive
```

```js
// ./minista.config.js
import { pluginArchive } from "minista-plugin-archive"

export default {
  plugins: [pluginArchive()],
}
```
