# minista-plugin-bundle

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-bundle
```

```js
// ./minista.config.js
import { pluginBundle } from "minista-plugin-bundle"

export default {
  plugins: [pluginBundle()],
}
```
