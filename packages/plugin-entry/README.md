# minista-plugin-entry

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-entry
```

```js
// ./minista.config.js
import { pluginEntry } from "minista-plugin-entry"

export default {
  plugins: [pluginEntry()],
}
```
