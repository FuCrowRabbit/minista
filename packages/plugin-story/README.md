# minista-plugin-story

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-story
```

```js
// ./minista.config.js
import { pluginStory } from "minista-plugin-story"

export default {
  plugins: [pluginStory()],
}
```
